## wanted-pre-onboarding-backend

### 지원자: 정현석

### 애플리케이셜 실행 방법

1. 레포지토리 클론
2. application.yml 작성 `src/main/resources/application.yml`

    ```yaml
    spring:
      datasource:
        driver-class-name: com.mysql.cj.jdbc.Driver
        url: jdbc:mysql:{{DB_URL}}
        username: {{DB_USER}}
        password: {{DB_PASSWORD}}
      jpa:
        show-sql: true
        properties:
          hibernate:
            format_sql: true
        hibernate:
          ddl-auto: update
    jwt:
      secret: {{JWT_SECRET}}
    ```
3. 실행

    ```shell
    ./gradlew bootrun
    ```
